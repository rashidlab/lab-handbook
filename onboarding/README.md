# New Member Onboarding

Welcome to the Rashid Lab! Follow this checklist to get set up.

## Week 1: Accounts & Access

- [ ] UNC ONYEN activated
- [ ] GitHub account created and added to rashidlab organization
- [ ] Longleaf HPC access requested ([ITS Help](https://help.unc.edu))
- [ ] Teams workspace joined
- [ ] Added to lab calendar
- [ ] Building/office access arranged

## Week 1: Computing Setup

- [ ] Clone lab-computing repo and review guides
- [ ] Set up Longleaf environment (see lab-computing/longleaf/)
- [ ] Install R/RStudio with lab packages
- [ ] Configure Git on local machine
- [ ] Test GitHub SSH access

## Week 2: Orientation

- [ ] Read lab policies (this repo)
- [ ] Review coding standards
- [ ] Meet with each lab member (15 min each)
- [ ] Attend first lab meeting
- [ ] Schedule regular 1:1 with PI

## Week 2: Project Setup

- [ ] Create project repo from template
- [ ] Set up package management/virtual environment
- [ ] Review example projects from other lab members
- [ ] Begin initial reading/literature review

## First Month

- [ ] Complete required training (CITI, etc.)
- [ ] Present brief introduction at lab meeting
- [ ] Draft initial project plan with PI
- [ ] Establish regular work schedule
