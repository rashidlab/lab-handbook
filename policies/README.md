# Lab Policies

## Communication

- **Teams**: Primary day-to-day communication
- **Email**: External communication, formal matters
- **GitHub**: Code reviews, technical discussions

## Meetings

- **Lab meetings**: Biweekly Thursdays, 9:00 AM
- **1:1 meetings**: Scheduled individually with PI

## Code & Data

- All code must be version controlled in GitHub
- Never commit sensitive data or credentials
- Use `.gitignore` templates provided in lab templates
- Document all code with comments and README files

## Publications

- All publications require PI review before submission
- Code and data must be prepared for public release
- Preprints encouraged for methodology papers

## Computing

- Use Longleaf for computationally intensive work
- Follow HPC best practices in lab-computing repo
- Request resources appropriately (don't over-request)

## Time Off

- Notify PI at least 2 weeks in advance for planned absences
- Keep calendar updated
- Arrange coverage for any ongoing responsibilities
